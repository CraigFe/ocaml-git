Unix implementation

The default implementation for Unix fixes the choice of Git.Inflate.S and Git.SHA.DIGEST by using camlzip's zlib bindings and nocrypto's SHA1 implementation.

It also uses Lwt_unix and Lwt_io for the IO functions.
